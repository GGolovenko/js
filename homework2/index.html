4. Should work for any object: 
const a = {};
 a.setName('John');
 alert(a.name); // 'John'
Answer: global/base Object needs a prototype for setName 
Object.prototype.setName = function(name) {
    this.name = name;
};

a.setName('John');

10. Why it works so and how to fix: 
var userJohn = {name: 'John', age: 30};
var userMike = userJohn;
userMike.name = 'Mike';
console.log(userMike.name) // Mike - correct behavior
console.log(userJohn.name) // Mike - wrong behavior, 'John' expected
Answer: object copied by reference and we have to clone it to fix the error. 
var userJohn = {name: 'John', age: 30};
var userMike = {...userJohn};
// Classic solution

userMike.name = 'Mike';
